## 使用说明

- 每个版本的更新内容、修复项和升级说明都写在对应的 GitHub Releases 页面。
- 应用内更新会直接使用本仓库中的 APK 下载地址。
- 首次通过浏览器或应用内更新安装 APK 时，Android 系统可能会提示允许“安装未知来源应用”或“允许来自此来源的应用”。按系统提示授权后，再继续安装即可。

## 版本说明约定
